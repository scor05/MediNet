# MediNet
Sistema de conexiones que permite una manera más fácil de agendar y manejar citas para doctores de varias clínicas y para usuarios que deseen buscar citas (y agendarlas a la vez)
Este proyecto se concentra principalmente en solventar las discrepancias que existen en todo el proceso de agendar y coordinar citas médicas, desde el lado de los doctores
hasta el lado de los pacientes. Lo que busca principalmente este proyecto es brindarle a los doctores, pacientes y secretarias una manera única para observar sus citas médicas,
con cada cambio siendo procesado en tiempo real, asímismo proveyendo varias comodidades a los pacientes para facilidad de comunicación, almacenamiento de datos médicos y demás
información que sea relevante al doctor/clínica que desee utilizar.
